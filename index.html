Bienvenido.

Mi nombre es Angelo Ormeño, soy desarrollador Web con sede en Perú, Lima, Callao. He trabajado en una amplia variedad de proyectos front-end y Back-end, desde aplicaciones para escritorio hasta plataformas de capacitación y auditoría de Gestión de Seguridad, con un enfoque en crear interfaces limpias y bien diseñadas que no solo se vean bien, sino que también ofrezcan una experiencia de usuario fluida.

Mi Trabajo

Una colección de proyectos en los que he trabajado.

repositorio

Domposer es un creador de sitios web con una interfaz que permite agregar y reorganizar HTML y CSS rápidamente, subir y descargar proyectos y archivos creados, iniciar sesión para guardar hasta 5 proyectos y puede instalarse como una aplicación web progresiva. La pila es Laravel para el backend/API y JavaScript estándar para el frontend. El repositorio de Github es privado actualmente.
